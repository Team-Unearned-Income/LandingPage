# Knock-in Landing Page

## 배포 URL

https://team-unearned-income.github.io/LandingPage/

## SEO 구성

- `index.html`: title, description, canonical, Open Graph, Twitter Card, JSON-LD structured data
- `robots.txt`: 검색엔진 크롤링 허용 및 sitemap 경로 제공
- `sitemap.xml`: 배포 URL과 최종 수정일 제공
- `assets/og-image.png`: SNS 공유용 1200x630 이미지
